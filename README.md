# EDOT Platform 🎓
A comprehensive Educational System for students, instructors, and parents.

## 🚀 Features
- **Intelligence Hub:** Analytics for student performance.
- **Library Module:** Digital resource management with PDF support.
- **Role-Based Dashboards:** Unique views for Admin, Teacher, and Supporter.
- **Agenda System:** Global calendar and task tracking.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express, MongoDB
